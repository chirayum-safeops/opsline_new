

# OpsLine — DevOps Consulting Website

A single-page, high-conversion consulting website for **OpsLine** with an electric blue (#0069FF) brand accent on a clean white background. The site is designed to impress technical decision-makers and drive meeting bookings.

---

## 1. Animated Background System
- Canvas-based particle system with infrastructure-like connecting lines that react to scroll position
- Subtle blue glow effects (#0069FF) that follow the user through sections
- Continuous, seamless animation across all sections — no visual breaks

## 2. Smart Navbar
- Clean white navbar with OpsLine logo
- Smooth scroll navigation links to each section
- Transforms on scroll: gains subtle shadow and slight background blur
- "Book a Strategy Session" CTA button always visible (electric blue)

## 3. Interactive Hero Section
- Bold headline: tagline about DevOps excellence
- Subtext with a clear value proposition
- Behind the text: a simulated DevOps dashboard showing:
  - **Moving CI/CD pipeline stages** (build → test → deploy) with animated progress
  - **Scrolling terminal logs** that look like real deployment output
  - **Live-updating cloud metrics** (CPU, latency, deployments) with animated charts
- Primary CTA: "Get a 20-min OpsLine Strategy Session"

## 4. Services Section
- 4 service cards with placeholder DevOps services:
  - CI/CD Pipelines
  - Cloud Infrastructure
  - Monitoring & Observability
  - Security & Compliance
- Each card has a soft blue glow on hover and a subtle floating animation
- Minimal icons with smooth entrance animations on scroll

## 5. "How OpsLine Works" — Visual Process Section
- 4-step process with progressive scroll reveal:
  1. Audit Infrastructure
  2. Automate Pipelines
  3. Secure Delivery
  4. Scale Safely
- Connected timeline with animated lines between steps
- Each step icon animates in sequence as user scrolls
- Brief description under each step

## 6. Case Studies / Results Section
- 2-3 placeholder result cards showing metrics (e.g., "40% faster deployments", "99.9% uptime achieved")
- Cards with subtle float animation and blue accent borders
- Clean, data-driven layout that builds trust

## 7. Client Testimonials & Logos
- Placeholder client logo bar (horizontally scrolling)
- 2-3 testimonial quotes in elegant cards
- Subtle fade-in on scroll

## 8. FAQ Section
- Accordion-style FAQ with common DevOps consulting questions
- Smooth expand/collapse animations
- Questions like "What does a DevOps audit include?", "How long does onboarding take?", etc.

## 9. Calendly Booking CTA
- A prominent section near the bottom with the CTA: "Get a 20-min OpsLine Strategy Session"
- Clicking opens a **Calendly modal** (inline embed) without leaving the page
- Uses the provided Calendly widget code pointing to `https://calendly.com/d/cx9m-tbp-2qs/opsline-meeting`
- Calendly badge widget also floats in the bottom-right corner throughout the site

## 10. Footer
- Company info, quick nav links, social placeholders
- Clean, minimal design consistent with the white theme

## Micro-Animations Throughout
- Cards float slightly with a subtle up-down motion
- Hover states have soft blue glow transitions
- Scroll-triggered fade-in and slide-up for all sections
- Smooth, professional — never over-the-top
